Introduction on VTU SGPA/CGPA Calculator


This VTU SPGA/CGPA Calculator was developed to ease the calculation of SPGA/CGPA for VTU students without any hassle. This SPGA/CGPA calculator can be used by students of 2018 scheme onwards.


SGPA stands for Semester Grade Point Average. It is used in educational institutes to measure the performance of a student in particular semester


CGPA stands for Cumulative Grade Point Average. It is used in ducational institutes to measure overall academic performance of a student.


The VTU SPGA/CGPA Calculator has the following :


  • SGPA Calculator
  • CGPA Calculator
  • CGPA to Percentage Calculator

The VTU SPGA/CGPA Calculator has the following features :


  • A Calculate button to calculate SPGA/CGPA.
  • A Reset button to clear all fields.
  • A Popup to display the result.
